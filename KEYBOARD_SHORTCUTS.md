# 快捷键配置指南

## 🎯 问题说明

### 为什么 Ctrl+Shift+S 不工作？

**原因**：`Ctrl+Shift+S` 是 Chrome 浏览器的内置快捷键，用于"保存页面"功能。Chrome Extension 的快捷键无法覆盖浏览器的内置快捷键。

### 解决方案

我们已将快捷键更改为不冲突的组合：

| 操作 | Windows/Linux | Mac | 说明 |
|------|--------------|-----|------|
| 保存选中文本 | `Alt+Shift+S` | `Cmd+Shift+D` | 已避免冲突 |
| 插入片段 | `Ctrl+Shift+V` | `Cmd+Shift+V` | 无冲突 |

## 🔧 如何自定义快捷键

如果你想使用其他快捷键组合，可以按以下步骤自定义：

### 步骤 1：打开快捷键设置

1. 在 Chrome 地址栏输入：`chrome://extensions/shortcuts`
2. 或者：
   - 打开 `chrome://extensions/`
   - 点击左上角的 ☰ 菜单
   - 选择"键盘快捷键"

### 步骤 2：找到 QA sidePanel

在页面中找到 "QA sidePanel" 部分，你会看到两个命令：
- **保存选中文本为片段**
- **插入片段到当前页面**

### 步骤 3：设置快捷键

1. 点击命令右侧的铅笔图标 ✏️
2. 按下你想要的快捷键组合
3. 点击"确定"保存

### 步骤 4：测试快捷键

1. 打开任意网页
2. 选中一些文本
3. 按下你设置的快捷键
4. 应该会打开保存对话框

## 💡 推荐的快捷键组合

### 不容易冲突的组合

**Windows/Linux**：
- `Alt+Shift+S` - 保存（默认）
- `Alt+Shift+Q` - 保存（备选）
- `Alt+S` - 保存（简单）
- `Ctrl+Alt+S` - 保存（备选）

**Mac**：
- `Cmd+Shift+D` - 保存（默认）
- `Cmd+Option+S` - 保存（备选）
- `Ctrl+Cmd+S` - 保存（备选）

### 避免使用的组合

以下快捷键已被 Chrome 占用，**不要使用**：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存页面 |
| `Ctrl+Shift+S` | 保存页面 |
| `Ctrl+P` | 打印 |
| `Ctrl+T` | 新标签页 |
| `Ctrl+W` | 关闭标签页 |
| `Ctrl+N` | 新窗口 |
| `Ctrl+Shift+N` | 无痕模式 |
| `Ctrl+Tab` | 切换标签页 |

## 🎨 快捷键使用技巧

### 技巧 1：右键菜单作为备选

如果快捷键不方便使用，可以使用右键菜单：
- 选中文本 → 右键 → "保存选中文本为片段"
- 在输入框右键 → "插入片段"

### 技巧 2：记忆快捷键

**助记方法**：
- `Alt+Shift+S` - **S**ave（保存）
- `Ctrl+Shift+V` - Paste/**V**（粘贴/插入）

### 技巧 3：一键操作流程

**保存流程**（3 秒内完成）：
```
1. 选中文本
2. Alt+Shift+S
3. 输入标题（自动检测语言）
4. Enter 保存
```

**插入流程**（2 秒内完成）：
```
1. 光标定位到输入框
2. Ctrl+Shift+V
3. 搜索或选择片段
4. Enter 插入
```

## 🔍 快捷键故障排查

### 问题 1：快捷键完全没反应

**检查清单**：
- [ ] 扩展是否已启用？访问 `chrome://extensions/`
- [ ] 是否在特殊页面？（chrome://、edge:// 等页面不支持）
- [ ] 是否有其他扩展使用了相同快捷键？
- [ ] 重新加载扩展试试

### 问题 2：快捷键偶尔不工作

**可能原因**：
- Content Script 还未加载完成
- 页面焦点不在正确的元素上
- 页面有自己的快捷键监听

**解决方案**：
- 刷新页面重试
- 使用右键菜单作为备选
- 等待页面完全加载后再使用

### 问题 3：Mac 上快捷键不同

**说明**：
- Mac 使用 `Cmd` 键代替 `Ctrl`
- Mac 使用 `Option` 键代替 `Alt`
- 默认配置已针对 Mac 优化

### 问题 4：快捷键与其他软件冲突

**解决方案**：
1. 在 `chrome://extensions/shortcuts` 自定义快捷键
2. 选择不与其他软件冲突的组合
3. 建议使用 `Alt+Shift+` 组合（较少被占用）

### 已知限制

1. **特殊页面不支持**：
   - `chrome://` 页面
   - `edge://` 页面
   - Chrome Web Store 页面
   - 扩展管理页面

2. **部分网站可能冲突**：
   - Gmail（有大量自定义快捷键）
   - Google Docs（键盘快捷键很多）
   - VS Code Web（IDE 快捷键）

**解决方案**：在这些网站使用右键菜单

## 🎯 最佳实践

### 1. 习惯养成

建议使用快捷键的场景：
- ✅ 浏览技术文档时保存代码
- ✅ 在编辑器中快速插入模板
- ✅ 在 JIRA/Confluence 中插入常用文本

**高效组合**：
1. 右手鼠标选中文本
2. 左手按 `Alt+Shift+S`
3. 输入标题，按 Enter
4. 继续工作，无需切换窗口

如果团队使用统一的快捷键：
1. 在团队文档中标注推荐的快捷键设置
2. 新成员入职时配置统一快捷键
3. 定期分享使用技巧
